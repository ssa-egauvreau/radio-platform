# Bundled console / handset sounds

Place WAV files here so the web console and Android app can play them before any agency upload.

| File | Used for |
|------|-----------|
| `ptt_permit.wav` | Talk permit |
| `channel_switch.wav` | Channel change |
| `emergency.wav` | Emergency alerts |
| `busy.wav` | Channel busy |
| `marker_1033.wav` | **10-33** channel marker on dispatch (12 s loop) |
| `volume.wav` | Handset volume-check key |

Copy your Motorola / Zello files per `docs/custom-radio-sounds.md`.
